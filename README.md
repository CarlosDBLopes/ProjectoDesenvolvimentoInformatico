🛒 DespensaSmart
Gestão de inventário alimentar e combate ao desperdício. Projeto desenvolvido no âmbito da unidade curricular de Projeto e Desenvolvimento Informático (2025/2026).

📖 Sobre o Projeto
O projeto DespensaSmart surge para resolver o problema comum do desperdício alimentar doméstico. Esta aplicação móvel centraliza a gestão do inventário (despensa) e a lista de compras, fornecendo alertas visuais automáticos baseados nos prazos de validade e garantindo uma gestão ágil e fluida do dia a dia.

✨ Principais Funcionalidades

📦 Gestão de Despensa: Registo detalhado de produtos com fotografia, cálculo automático de validade (Verde, Amarelo, Vermelho) e controlo de stock.

📝 Lista de Compras: Criação de listas dinâmicas com a possibilidade de marcar itens como comprados e transferi-los para a despensa.

📊 Dashboard Analítico: Painel visual com gráfico de percentagem de validades e tabelas de acesso rápido a itens expirados.

📷 Leitura de Código de Barras: Inserção ágil de produtos através de scanning utilizando a câmara do dispositivo.

🌍 Internacionalização (i18n): Interface preparada para múltiplos idiomas.

🛠️ Tecnologias Utilizadas

📱 Frontend: React Native com framework Expo

☁️ Backend & Base de Dados: Supabase (PostgreSQL, Storage e Auth)

💻 Linguagem: TypeScript

🎨 Interface: Componentes nativos, Modais e animações

⚙️ Bibliotecas Extra: react-native-vision-camera, react-native-toast-message

🚀 Como Executar Localmente (Para Avaliação)

Passo 1: Certifique-se de que tem o Node.js instalado no computador e a aplicação Expo Go no seu telemóvel.

Passo 2: Faça o clone do repositório para a sua máquina (git clone https://github.com/CarlosDBLopes/ProjectoDesenvolvimentoInformatico.git).

Passo 3: Abra a pasta do projeto no terminal e instale as dependências (npm install).

Passo 4: Crie um ficheiro .env.local na raiz do projeto e insira as variáveis EXPO_PUBLIC_SUPABASE_URL e EXPO_PUBLIC_SUPABASE_ANON_KEY (as chaves exatas foram fornecidas no relatório do projeto).

Passo 5: Inicie o servidor local executando o comando npx expo start.

Passo 6: Abra a aplicação Expo Go no telemóvel e faça scan do QR Code apresentado no terminal.
